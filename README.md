
# Pre_SE_Project

โปรเจคนี้เป็นโปรเจคที่พัฒนาเว็ปแอพพิเคชั่นสำหรับยื่นคำร้อง

ภาษา และ Framework ที่เกี่ยวข้อง

    1. HTML & CSS
    2. javascript
    3. AngularJS
    4. NodeJS
    5. MySQL
